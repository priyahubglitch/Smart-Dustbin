1. Components Required
• Arduino Uno
• Breadboard
• Ultrasonic Sensor (HC‑SR04)
• Servo Motor (SG90)
• Jumper Wires (Male–Male, Male–Female)
• External Power (optional for large servos)
2. Circuit Connections
2.1 Breadboard Power Setup
• Connect Arduino 5V to the breadboard + (red) rail.
• Connect Arduino GND to the breadboard – (blue) rail.
This powers all components through the breadboard.
2.2 Ultrasonic Sensor (HC-SR04)
• VCC → Breadboard + (5V)
• GND → Breadboard – (GND)
• TRIG → Arduino Digital Pin 9
• ECHO → Arduino Digital Pin 10
2.3 Servo Motor (SG90)
• Brown/Black (GND) → Breadboard –
• Red (VCC) → Breadboard +
• Orange/Yellow (Signal) → Arduino Pin 3
